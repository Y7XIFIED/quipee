# Y7XIFIED Portfolio - Powered by Y7X

This is a high-performance portfolio website built with React, Vite, and Tailwind CSS, featuring a "chaotically logical" tech-obsessed persona and advanced animations.

## Project Overview

- **Name**: Y7XIFIED
- **Tagline**: Powered by Y7X
- **Persona**: Highly caffeinated, tech-obsessed, curiosity-driven.
- **Hero Tagline**: "Turning caffeine into code and chaos."
- **Style**: Minimalist, High Contrast, Glitchy, Motion-heavy.
- **Tech Stack**: React 18, TypeScript, Vite, Tailwind CSS, Framer Motion, Lenis, Matter.js.

## Animation Suite (50+ Upgrades)

### Global Effects
1.  **Smooth Scrolling**: Powered by Lenis.
2.  **Custom Cursor**: Magnetic, blend-mode enabled.
3.  **Noise Overlay**: Cinematic grain texture.
4.  **Scroll Progress**: Minimal top bar.
5.  **Preloader**: 0-100% count-up.
6.  **Console Easter Egg**: Hidden message for developers.
7.  **Click Particles**: Visual explosions on click.
8.  **Hacker Mode**: Matrix rain overlay (Konami Code triggered).
9.  **Coffee Mode**: Visual fill effect.
10. **Grid Background**: Interactive warping grid.
11. **Spotlight**: Mouse flashlight effect.
12. **Achievements**: Gamified toast notifications.
13. **Terminal Modal**: Full-screen CLI (Cmd+K).

### Section-Specific Effects
14. **Hero Glitch**: Cyberpunk text scrambling.
15. **Typewriter Effect**: Self-correcting typing animation in Hero.
16. **Physics Playground**: Falling icons triggered by "Poke the System".
17. **Parallax Background**: Depth effects.
18. **Marquee**: "Goofy Facts" scrolling band in Philosophy.
19. **Tech Stack Marquee**: Infinite scrolling logos.
20. **Code Showcase**: Self-typing code block.
21. **Draggable Traits**: Physics-enabled draggable cards in Traits section.
22. **Hover Image Reveal**: Floating gradient reveal.
23. **System Status**: Live "Brain CPU" monitor in Footer.
24. **Curtain Footer**: Fixed footer reveal.

## Design System

### Colors
- **Dark**: `#050505` (Background)
- **Light**: `#F4F4F0` (Philosophy Section)
- **Accent**: `#D4F268` (Lime Green)
- **Text**: White on Dark, Black on Light

### Typography
- **Font**: Inter (Sans-serif) & Monospace for code/tech vibes.
